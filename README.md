# NFT Minter Machine

A simple ERC721 minter machine that implemented by ERC721_extended contract.

Feature:
- Wallet support: Metamask, WalletConnect
- Mint random token id
- Whitelist 
- Provenance Hash 
- Reveal and Not reveal option, something similar to gacha system
